# 🌳 Welcome to Boemly

![Boemly logo](https://cdn.tree.ly/assets/v3/boemly-banner@2x.webp)

Boemly is the open source component library maintained by [Tree.ly](https://tree.ly) based on
[ChakraUI](https://chakra-ui.com/).

## 🏃‍♀️ Getting started

Install Boemly:

```bash
npm i boemly --save
```

Include the theme provider in the root of your app and start using the Boemly components:

```tsx
import { Button, BoemlyThemeProvider } from 'boemly';

function App() {
  return (
    <BoemlyThemeProvider>
      <Button>Button</Button>
    </BoemlyThemeProvider>
  );
}

export default App;
```
