## Arguments

### `<Table.Root>`

| Property  | Values                                | Default    |
| --------- | ------------------------------------- | ---------- |
| `variant` | `'simple'`, `'striped'`, `'unstyled'` | `'simple'` |
| `size`    | `'sm'`, `'md'`, `'lg'`                | `'md'`     |

### `<Table.Cell>` and `<Table.ColumnHeader>`

| Property    | Values          | Default |
| ----------- | --------------- | ------- |
| `isNumeric` | `true`, `false` | `false` |
