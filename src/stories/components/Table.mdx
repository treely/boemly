## Arguments

### `<Table.Root>`

| Property   | Values                 | Default   |
| ---------- | ---------------------- | --------- |
| `variant`  | `'line'`, `'outline'`  | `'line'`  |
| `size`     | `'sm'`, `'md'`, `'lg'` | `'md'`    |
| `striped`  | `'true'`, `'false'`    | `'false'` |
| `unstyled` | `'true'`, `'false'`    | `'true'`  |
